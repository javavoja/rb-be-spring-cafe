#### 프로젝트 개요

자바 언어와 스프링 프레임워크를 사용해 카페 사이트의 백엔드 부분을 단계별로 구현하는 미션을 수행합니다. <br> 
네트워크와 스프링 등 새로운 지식을 학습할 수 있습니다. 학습 내용을 정리해 링크를 첨부합니다. <br>
초기 상태를 구성하며 gradle 빌드 과정과 프로젝트 구조에 관해 추가로 학습했습니다.<br>
